# Retrieval Augmented Generation Workshop

A workshop registration landing page for Singleton's Career Center.

## Overview

This is a single-page registration site for the **Retrieval Augmented Generation (RAG) Workshop**. It includes workshop details, learning outcomes, target audience information, and a registration form.

## Files

- `index.html` - Main page content
- `style.css` - Styling and layout

## Features

- Responsive design
- Clean, modern blue gradient styling
- Workshop information sections
- Simple registration form

## How to Use

1. Open `index.html` in a browser
2. Edit `index.html` to update workshop details
3. Edit `style.css` to customize colors and styling

## Design Elements

- **Color Scheme**: Professional blue (#0066cc)
- **Headings**: Centered, bold, 2rem (h1) and 1.5rem (h2)
- **Lists**: Centered with inside list markers
- **Button**: Blue background with hover effect and lift animation
- **Layout**: Two-column on desktop, stacked on mobile
